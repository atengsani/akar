1 Pengertian Bentuk Akar

Bentuk akar adalah suatu bentuk bilangan yang dinyatakan dalam tanda akar (‚àö). Bentuk akar merupakan cara lain untuk menuliskan bilangan berpangkat pecahan.

Definisi Matematis:

$$\sqrt[n]{a} = a^{\frac{1}{n}}$$

Dimana:
‚Ä¢ $n$ = indeks akar (bilangan bulat positif)
‚Ä¢ $a$ = bilangan yang diakarkan (radicand)
‚Ä¢ $\sqrt[n]{a}$ = akar ke-n dari a

2 Jenis-jenis Bentuk Akar

üîπ Akar Kuadrat

$$\sqrt{a} = \sqrt[2]{a} = a^{\frac{1}{2}}$$

Contoh:

‚Ä¢ $\sqrt{9} = 3$

‚Ä¢ $\sqrt{16} = 4$

‚Ä¢ $\sqrt{25} = 5$

üî∏ Akar Kubik

$$\sqrt[3]{a} = a^{\frac{1}{3}}$$

Contoh:

‚Ä¢ $\sqrt[3]{8} = 2$

‚Ä¢ $\sqrt[3]{27} = 3$

‚Ä¢ $\sqrt[3]{64} = 4$

üî∂ Akar ke-n

$$\sqrt[n]{a} = a^{\frac{1}{n}}$$

Contoh:

‚Ä¢ $\sqrt[4]{16} = 2$

‚Ä¢ $\sqrt[5]{32} = 2$

‚Ä¢ $\sqrt[6]{64} = 2$

üî∑ Bentuk Akar Campuran

$$a\sqrt{b}$$

Contoh:

‚Ä¢ $2\sqrt{3}$

‚Ä¢ $5\sqrt{2}$

‚Ä¢ $3\sqrt{7}$

3 Sifat-sifat Bentuk Akar

üìã Sifat-sifat Dasar:

1. Perkalian akar: $$\sqrt{a} \times \sqrt{b} = \sqrt{a \times b}$$
2. Pembagian akar: $$\frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}}$$
3. Akar dari pangkat: $$\sqrt{a^2} = |a|$$
4. Pangkat akar: $$(\sqrt{a})^2 = a$$

4 Contoh Soal dan Pembahasan

üìù Contoh 1:

Soal: Sederhanakan $\sqrt{18}$

Penyelesaian:

$\sqrt{18} = \sqrt{9 \times 2}$

$= \sqrt{9} \times \sqrt{2}$

$= 3\sqrt{2}$

üìù Contoh 2:

Soal: Hitunglah $\sqrt{12} + \sqrt{27}$

Penyelesaian:

$\sqrt{12} = \sqrt{4 \times 3} = 2\sqrt{3}$

$\sqrt{27} = \sqrt{9 \times 3} = 3\sqrt{3}$

$\sqrt{12} + \sqrt{27} = 2\sqrt{3} + 3\sqrt{3} = 5\sqrt{3}$
