1 Pengertian Bentuk Akar

Bentuk akar adalah suatu bentuk bilangan yang dinyatakan dalam tanda akar (√). Bentuk akar merupakan cara lain untuk menuliskan bilangan berpangkat pecahan.

Definisi Matematis:

$$\sqrt[n]{a} = a^{\frac{1}{n}}$$

Dimana:
• $n$ = indeks akar (bilangan bulat positif)
• $a$ = bilangan yang diakarkan (radicand)
• $\sqrt[n]{a}$ = akar ke-n dari a

2 Jenis-jenis Bentuk Akar

🔹 Akar Kuadrat

$$\sqrt{a} = \sqrt[2]{a} = a^{\frac{1}{2}}$$

Contoh:

• $\sqrt{9} = 3$

• $\sqrt{16} = 4$

• $\sqrt{25} = 5$

🔸 Akar Kubik

$$\sqrt[3]{a} = a^{\frac{1}{3}}$$

Contoh:

• $\sqrt[3]{8} = 2$

• $\sqrt[3]{27} = 3$

• $\sqrt[3]{64} = 4$

🔶 Akar ke-n

$$\sqrt[n]{a} = a^{\frac{1}{n}}$$

Contoh:

• $\sqrt[4]{16} = 2$

• $\sqrt[5]{32} = 2$

• $\sqrt[6]{64} = 2$

🔷 Bentuk Akar Campuran

$$a\sqrt{b}$$

Contoh:

• $2\sqrt{3}$

• $5\sqrt{2}$

• $3\sqrt{7}$

3 Sifat-sifat Bentuk Akar

📋 Sifat-sifat Dasar:

1. Perkalian akar: $$\sqrt{a} \times \sqrt{b} = \sqrt{a \times b}$$
2. Pembagian akar: $$\frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}}$$
3. Akar dari pangkat: $$\sqrt{a^2} = |a|$$
4. Pangkat akar: $$(\sqrt{a})^2 = a$$

4 Contoh Soal dan Pembahasan

📝 Contoh 1:

Soal: Sederhanakan $\sqrt{18}$

Penyelesaian:

$\sqrt{18} = \sqrt{9 \times 2}$

$= \sqrt{9} \times \sqrt{2}$

$= 3\sqrt{2}$

📝 Contoh 2:

Soal: Hitunglah $\sqrt{12} + \sqrt{27}$

Penyelesaian:

$\sqrt{12} = \sqrt{4 \times 3} = 2\sqrt{3}$

$\sqrt{27} = \sqrt{9 \times 3} = 3\sqrt{3}$

$\sqrt{12} + \sqrt{27} = 2\sqrt{3} + 3\sqrt{3} = 5\sqrt{3}$
